Add Weather Location

This guide will show you how to add a static location to the default weather add-on (Gismeteo) in aNZac Build. There are many weather add-ons available for Kodi but Gismeteo is used as the default in aNZac Build as it supports automatic geo-location by IP address although this can sometimes be inaccurate.

Step A
Navigate DOWN to 'Settings' and RIGHT to 'Weather' and press OK (pictured left)

Step B
Navigate to 'Gismeteo' and press OK
(this should already be selected)

Step C
Navigate to 'CONFIGURE' and press OK
(this should already be selected)

Step D
Navigate RIGHT to 'Current Location' and press OK to switch 'Off'
(this disables automatic location)

Step E
Navigate DOWN to 'Location 1' and press OK

Step F
Type in the name of the location you want to use then navigate to 'Done' and press OK.
(in this example we selected 'picton')

Step G
You will now see a list of locations that matched 'picton'. Navigate to the correct one and press OK

Step H
Navigate RIGHT to 'OK' and press OK
Note: Press BACK twice to return to the 'Main Menu'. The new settings will not take effect until Kodi is restarted.
