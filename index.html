aNZac Build

a Kodi build for the lands down under

About

aNZac Build is a collection of third party Kodi media player add-ons that search the Internet and provide links to on-line content that is publicly available. It provides a special focus on content from the lands down under (New Zealand and Australia) hence the name. It is compatible with both Android 6 or higher TV boxes and Phones and the LibreELEC operating system. There are versions for both Kodi 17 Krypton and Kodi 18 Leia available (please see Versions below).

Instillation

To install aNZac Build on your Android 6 + or LibreELEC device follow the steps below. (you will need to connect your device to the Internet)
1) Go to Settings then File Manager and then Add Source and enter https://anzacbuild.github.io/anzac and OK then across to OK and OK again.
2) Go Back and to Addons and Install From Zip and then go to anzac and then to plugin.anzactools and hit OK (you will need to enable unknown sources if prompted)
3) Go to My Addons then Program Addons then aNZac Tools and OK then Open and OK (you should be prompted from here to the end)
4) Go to Builds and select the build for your system. (Do not pick the wrong version or it will not work)
5) Go to Fresh Install or Standard Install and OK. (Fresh install will reset Kodi before installing. Standard will keep your current settings etc)
6) Once complete select Force Close to complete the instillation.
